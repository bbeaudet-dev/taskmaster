# taskmaster

A personal task manager and life-automation system: task list, reminders,
NFC-triggered physical actions, optional app/game lockdown, and daily
summaries — reachable via chat and Apple Shortcuts.

Backend and scheduling are owned by [Convex](https://convex.dev); the mobile
app is React Native + Expo with small native Swift App Extensions for Apple's
App Intents and Screen Time APIs.

See [`docs/START HERE.md`](docs/START%20HERE.md) for the current project
briefing.
